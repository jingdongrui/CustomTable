# 表格组件使用说明

本项目实现了一个基于 Vue 3 + TypeScript + Element Plus 的表格组件，支持多种自定义功能，优化大数据量加载体验。

## 📂 **模拟数据**
模拟数据存放于 `模拟数据.xlsx`。

## ✨ **功能特点**
1. **支持多级表头**
2. **自定义固定列**
3. **拖动行变更位置**
4. **自定义表格**
5. **可控制行为多选或单选**
6. **可控制列有无序号**
7. **相同数据合并单元格**
8. **Excel 数据导入**
9. **支持虚拟滚动表格**
